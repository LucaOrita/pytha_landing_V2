# Referinta API si Export

PYTHA ofera multiple formate de export si optiuni de integrare cu sisteme externe.

## Formate de export 3D

| Format | Extensie | Utilizare |
|--------|----------|-----------|
| DWG/DXF | .dwg, .dxf | AutoCAD, schite 2D |
| STEP | .stp, .step | Schimb CAD universal |
| STL | .stl | Imprimare 3D |
| OBJ | .obj | Vizualizare, randare |
| 3DS | .3ds | Compatibilitate legacy |
| SKP | .skp | SketchUp |
| PDF | .pdf | Documentatie |

## Export CNC

| Format | Extensie | Masini compatibile |
|--------|----------|-------------------|
| G-Code | .nc, .gcode | Universal |
| MPR | .mpr | Homag WoodWop |
| CIX | .cix | Biesse bSolid |
| PGM | .pgm | Diverse |
| XML | .xml | Nesting, etichete |

## Export date

### CSV — Liste piese

Campuri disponibile:
- Dimensiuni (L, l, H)
- Material, cant, finisaj
- Cantitate, numar serie
- Nume program CNC
- Atribute personalizate

### XML — Etichete

Suport pentru:
- Coduri QR si barcode
- Sabloane personalizabile
- Integrare imprimante industriale

## Plug-in API

PYTHA ofera o interfata de programare deschisa pentru dezvoltarea de plug-in-uri custom. Contacteaza PYTHA Lab pentru documentatia dezvoltatorilor.
