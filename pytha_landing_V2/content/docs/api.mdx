## Formate de export 3D

PYTHA suporta export in cele mai utilizate formate CAD si vizualizare:

<Tabs items={['CAD Standard', 'Vizualizare', 'Schimb date']}>

<Tab value="CAD Standard">
| Format | Extensie | Utilizare |
|--------|----------|-----------|
| DWG | .dwg | AutoCAD, schite 2D/3D |
| DXF | .dxf | Schimb universal 2D |
| PDF | .pdf | Documentatie, planse tehnice |

Exportul DWG/DXF pastreaza layerele, cotarile si atributele din modelul PYTHA.
</Tab>

<Tab value="Vizualizare">
| Format | Extensie | Utilizare |
|--------|----------|-----------|
| STL | .stl | Imprimare 3D, prototipare |
| OBJ | .obj | Vizualizare, randare externa |
| 3DS | .3ds | Compatibilitate legacy |
| SKP | .skp | SketchUp |

Formatele de vizualizare exporta geometria cu materiale si texturi aplicate.
</Tab>

<Tab value="Schimb date">
| Format | Extensie | Utilizare |
|--------|----------|-----------|
| STEP | .stp, .step | Schimb CAD universal (ISO) |
| IGES | .igs | Schimb geometrie suprafete |

STEP este recomandat pentru colaborarea cu alte sisteme CAD (SolidWorks, Inventor, etc.).
</Tab>

</Tabs>

## Export date productie

### CSV — Liste piese

Campuri disponibile in exportul CSV:

<Accordions>

<Accordion title="Campuri standard">
- Dimensiuni (Lungime, Latime, Grosime)
- Material si decor
- Cant per muchie (tip, grosime)
- Cantitate
- Numar de serie / cod piesa
- Nume program CNC asociat
</Accordion>

<Accordion title="Campuri extinse">
- Atribute personalizate (definite de utilizator)
- Greutate calculata
- Pret material (daca e configurat)
- Pozitie in ansamblu (corp, nivel, zona)
- Directie fibra
</Accordion>

</Accordions>

## Plug-in API

PYTHA ofera o interfata de programare deschisa pentru dezvoltarea de plug-in-uri personalizate. Plug-in-urile pot:

- Automatiza operatii repetitive
- Crea generatoare custom de corpuri
- Integra cu sisteme ERP externe
- Genera rapoarte personalizate

> **Nota**: Contacteaza PYTHA Lab GmbH pentru documentatia dezvoltatorilor si SDK-ul de plug-in-uri.
