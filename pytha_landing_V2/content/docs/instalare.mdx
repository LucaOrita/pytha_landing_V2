# Instalare PYTHA

## Cerinte de sistem

- Windows 10/11 (64-bit)
- Procesor: Intel i5 / AMD Ryzen 5 sau superior
- RAM: minim 8GB (recomandat 16GB)
- GPU: dedicat cu suport OpenGL 4.0+
- Spatiu disc: 2GB pentru instalare

## Pasi de instalare

1. Descarca pachetul de instalare de pe [pytha.com](https://pytha.com)
2. Ruleaza installerul ca Administrator
3. Selecteaza modulele dorite
4. Activeaza licenta cu cheia primita

## Configurare initiala

Dupa instalare, configureaza:
- **Unitati de masura** — mm sau inch
- **Template proiect** — mobilier, arhitectura, sau personalizat
- **Conexiune CNC** — selecteaza postprocesorul potrivit masinii tale
