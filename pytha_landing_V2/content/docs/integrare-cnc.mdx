## Formate suportate

PYTHA exporta direct catre cele mai populare formate CNC industriale, fara conversii intermediare.

<Tabs items={['Homag', 'Biesse', 'Alte formate']}>

<Tab value="Homag">
### Homag / Weeke (WoodWop)

Format: **MPR** (.mpr)

Export direct compatibil cu WoodWop. Include:
- Coordonate gaurire (X, Y, Z, diametru, adancime)
- Frezari de contur si de buzunar
- Aplicare cant automat
- Referinte scule din biblioteca WoodWop

```
Masini compatibile:
- Homag Venture / Centateq
- Weeke Vantech / Optimat
- Homag Drillteq
```
</Tab>

<Tab value="Biesse">
### Biesse (bSolid)

Format: **CIX** (.cix)

Export nativ pentru bSolid/bNest. Include:
- Operatii complete de gaurire si frezare
- Informatii de cant per piesa
- Secvente de prelucrare optimizate

```
Masini compatibile:
- Biesse Rover
- Biesse Skipper
- Biesse Akron (cantuitoare)
```
</Tab>

## Configurare postprocesor

<Steps>

<Step>
### Selecteaza producatorul

Deschide **Settings → CAM Interface** si selecteaza producatorul masinii tale din lista.
</Step>

<Step>
### Configureaza parametrii

Seteaza parametrii specifici masinii:
- Viteze de avans si rotatie
- Biblioteca de scule (diametre, lungimi)
- Origini de referinta (stanga-jos, centru)
- Offset-uri specifice
</Step>

<Step>
### Testeaza cu un proiect simplu

Creeaza un corp de test cu cateva gauriri si o frezare de contur. Exporta si verifica fisierul in simulatorul masinii tale.
</Step>

<Step>
### Valideaza pe masina

Ruleaza primul program CNC pe o piesa de test (rest de material). Verifica pozitiile de gaurire si traiectele de frezare.
</Step>

</Steps>

## Flux de lucru complet

```
Design 3D (Basic)
    ↓
Workshop (atribuire operatii CNC)
    ↓
Nesting (optimizare taiere panouri)
    ↓
CAM Interface (export fisier CNC)
    ↓
Masina CNC (productie)
```

## Intrebari frecvente CNC

<Accordions>

<Accordion title="Pot folosi PYTHA cu masini second-hand?">
Da, atata timp cat masina accepta unul din formatele suportate (G-code fiind cel mai universal). Configureaza postprocesorul conform specificatiilor masinii.
</Accordion>

<Accordion title="Ce se intampla daca masina mea nu e in lista?">
Contacteaza echipa Adline — putem crea un postprocesor personalizat pentru masina ta. G-code universal functioneaza cu majoritatea masinilor CNC.
</Accordion>

<Accordion title="Export-ul include si informatii de cant?">
Da. Workshop-ul atribuie cantul pe fiecare muchie, iar informatia se exporta in fisierul CNC sau in fisierul CSV separat pentru cantuitoare automate.
</Accordion>

</Accordions>
