# Integrare CNC

PYTHA exporta direct catre cele mai populare formate CNC industriale.

## Formate suportate

- **WoodWop (.mpr)** — Homag, Weeke
- **bSolid (.cix)** — Biesse
- **AlphaCam** — Vero Software
- **G-Code** — universal
- **DDX** — EasyWood
- **SCM Maestro** — SCM Group

## Configurare postprocesor

1. Deschide Settings → CAM Interface
2. Selecteaza producatorul masinii
3. Configureaza parametrii specifici (viteze, scule)
4. Testeaza cu un proiect simplu

## Flux de lucru tipic

Design 3D → Workshop (imbinari) → Nesting (debitare) → Export CNC → Productie
